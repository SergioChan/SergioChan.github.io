POPULARITY - A widget that shows the trend of your new stars and followers on GitHub


POPULARITY

Intro

Popularity is a GitHub extension widget for you to trace the trends of your increasing stars and followers. You can easily view your data graph in the Today Extension. We already have graph view for commits data, as well as Contributions, (View Contributions for GitHub on App Store) but there isn't a widget or tool to manage or trace our Popularity on GitHub. :-)
